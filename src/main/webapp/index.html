Recurso Cliente JAX-RS REST

Serviço getLista

URL de acesso: rest/cliente

Comando:curl http://localhost:8080/webservice_cliente_jaxrs_rest/rest/cliente

Serviço getCliente(string)

URL de acesso: rest/cliente/1

Comando:curl http://localhost:8080/webservice_cliente_jaxrs_rest/rest/cliente/1

Serviço inserir(Cliente)

Comando:curl -v -X POST -H "Content-Type: application/json" -d "{\"clienteId\":\"3\",\"nome\":\"Pedro\",\"cpf\":\"45678912399\"}" http://localhost:8080/webservice_cliente_jaxrs_rest/rest/cliente

Serviço alterar(Cliente)

Comando:curl -v -X PUT -H "Content-Type: application/json" -d "{\"clienteId\":\"3\",\"nome\":\"Antônio\",\"cpf\":\"2222222222\"}" http://localhost:8080/webservice_cliente_jaxrs_rest/rest/cliente

Serviço excluir(clienteId)

Comando:curl -v -X DELETE http://localhost:8080/webservice_cliente_jaxrs_rest/rest/cliente/3
